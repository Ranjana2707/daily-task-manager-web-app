# Daily Task Manager

A beginner-friendly web app to add, complete, and delete daily tasks. Tasks are saved in your browser using LocalStorage, so they remain after refresh.

## Features
- **Add tasks:** Quickly add a new task via the input and Add button.
- **Mark completed:** Toggle completion with a checkbox.
- **Delete tasks:** Remove tasks you no longer need.
- **Persistent storage:** Tasks are stored in LocalStorage.
- **Clean UI:** Simple, accessible, and responsive interface.

## Technologies used
- **HTML:** Semantic structure for accessibility.
- **CSS:** Modern styling with variables, responsive layout.
- **JavaScript:** DOM manipulation, event delegation, LocalStorage.

## How to run the project
1. **Download or clone** this repository.
2. **Open `index.html`** in any modern browser (Chrome, Edge, Firefox).
3. **Start adding tasks**—they’ll auto-save in LocalStorage.
